# Shared Wokwi MicroPython modules

**`bmp180.py`** — I²C driver for `board-bmp180` (labs 4 and 5).

In Wokwi, create **File → New file → `bmp180.py`** and paste this module next to `main.py` and `diagram.json` from the lab folder (`lab04-i2c-sensor/` or `lab05-capstone/`).
